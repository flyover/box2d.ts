A TypeScript port of Box2D.

https://flyover.github.io/box2d.ts/Testbed/


## How to build it

```
git clone git@github.com:flyover/box2d.ts.git && cd box2d.ts
npm install
make build
```

## Run it

```
npx http-server -o
```
